# HDREZKA plugin для Lampa

Плагин-балансер, который добавляет HDREZKA как источник онлайн-просмотра прямо в карточку фильма/сериала Lampa и позволяет подключить ваш аккаунт `rezka.fi` (логин + пароль).

## Возможности

- 🔴 **Отдельная кнопка «HDREZKA» рядом со «Смотреть» / «Онлайн»** на странице фильма. Кнопка встаёт первой и сразу получает фокус — просто нажмите **OK** и сразу откроется экран просмотра.
- 🔐 **Вход по логину и паролю** через `POST /ajax/login/` (как на сайте). Сессия (`dle_user_id`, `dle_password`) сохраняется локально в `Lampa.Storage`.
- 🎬 **Балансер** в стандартной кнопке «Онлайн» — HDREZKA доступна и оттуда.
- 🎙 **Выбор озвучки (переводчика)** — все доступные переводы фильма попадают в фильтр «Перевод» в верхней части экрана.
- 📺 **Выбор качества** (360p / 480p / 720p / 1080p / 1080p Ultra / 4K) — передаётся в `Lampa.Player`, переключается прямо в плеере.
- 📅 **Сериалы**: сезоны и серии парсятся со страницы, с таким же выбором перевода.
- 📝 Субтитры (если есть в ответе HDREZKA) подхватываются и передаются в плеер.
- ⚡ **Клик вправо** на пульте открывает панель фильтров (перевод/сезон/качество/сортировка).
- 🖼 **Постеры отдельных серий** из TMDB (`still_path`), с фолбэком на общий постер сериала.
- ⚡️ **Мгновенные имена серий** при переключении сезона (TMDB-имена предзагружаются в фоне).
- 💾 **Запоминание качества**: выбор в плеере сохраняется в `rezka_quality` и используется при следующем запуске.
- 🎯 **Долгое нажатие зелёной кнопки HDREZKA** → меню выбора варианта (год/версия) с полем поиска. Идеально для одноимённых фильмов (Мумия 1999/2026 и т. п.).
- 🌐 **Настраиваемый домен**: по умолчанию `https://rezka.fi`, можно переключить на любое рабочее зеркало (`hdrezka.ag`, `rezka.ag` и т. д.).
- 🛰 Опциональная поддержка **CORS-прокси** на случай блокировок.

## Установка

1. Загрузите файл `rezka.js` на любой публично доступный HTTPS-хост (GitHub Pages, ваш сервер, gist в raw-режиме и т. п.). Запомните прямую ссылку, например:
   ```
   https://example.com/rezka.js
   ```
2. Откройте Lampa → **Настройки** → **Расширения** → **Добавить плагин**.
3. Вставьте URL, нажмите **Готово**, проверьте статус — должен быть `200`.
4. **Перезапустите** Lampa.

> ⚠️ Если ваша Lampa работает по `https://`, плагин и прокси тоже должны быть по `https://`. Иначе браузер (или Android WebView) заблокирует mixed-content.

## Подключение аккаунта HDREZKA

1. После установки в **Настройки** появится новая секция **HDREZKA**.
2. Заполните:
   - **Домен HDREZKA** — оставьте `https://rezka.fi` или укажите рабочее зеркало.
   - **Логин / E-mail** — логин или email от вашего аккаунта.
   - **Пароль** — пароль (хранится только на устройстве).
3. Нажмите **Войти в аккаунт**.
   - Описание этого пункта обновится:
     - 🟢 «Вы вошли в аккаунт» — успешно;
     - 🔴 «Ошибка: …» — что-то пошло не так (неверный пароль, капча, CORS).
4. Откройте любой фильм — рядом со «Смотреть» / «Онлайн» будет зелёная кнопка **HDREZKA**, уже под фокусом. Нажмите **OK** — откроется экран выбора озвучки / сезона / серии.

## Если автологин не работает: вход по Cookie (рекомендуется)

Lampa-Android использует нативный сетевой стек (OkHttp), который **не пробрасывает Set-Cookie из ответа в `document.cookie`**. Поэтому даже если `/ajax/login/` отвечает `success: true`, последующие запросы могут идти как «неавторизованные» и сервер вернёт страницу входа.

Надёжное решение — взять cookies из настольного браузера и вставить их вручную:

1. Откройте `https://rezka.fi/` в Chrome/Firefox на компьютере, войдите в свой аккаунт.
2. F12 → вкладка **Application** (или **Storage**) → **Cookies** → выберите `https://rezka.fi`.
3. Скопируйте значения двух cookie:
   - `dle_user_id`
   - `dle_password`
4. В Lampa → **Настройки → HDREZKA → Cookie (ручной вход)**, вставьте строку вида:
   ```
   dle_user_id=12345; dle_password=abcd1234ef5678gh
   ```
5. Нажмите **OK** — статус сменится на 🟢. Откройте любой фильм, кнопка HDREZKA сразу заработает.

> Cookie живёт месяцами. Когда закончится — повторите шаги 1-4.

## Если не работает: CORS-прокси

HDREZKA может ответить ошибкой CORS, потому что вы заходите с домена Lampa. Решений два:

- Использовать **Lampa без https** (HTTP-сборка) — у неё CORS обычно отключён.
- Указать в настройках плагина **CORS-прокси**. Любой `lampac`-совместимый прокси работает:
  ```
  https://your-proxy.example.com/
  ```
  Плагин будет вызывать `https://your-proxy.example.com/https://rezka.fi/...`.

## Безопасность

- Логин и пароль хранятся в `Lampa.Storage` локально на устройстве — на сторонние серверы не отправляются.
- Для входа используется только официальный endpoint `https://rezka.fi/ajax/login/`.
- Если вы используете публичный CORS-прокси — пароль пройдёт через него. Используйте только доверенные прокси, лучше — собственный.

## Что делать если домен `rezka.fi` не открывается

Откройте «Настройки → HDREZKA → Домен HDREZKA» и подставьте актуальное зеркало (см. список рабочих зеркал в Telegram-каналах HDREZKA, на сайте поддержки и т. п.).

## Известные ограничения

- **Premium-контент**: 4K и некоторые озвучки доступны только владельцам HDrezka Premium. Без премиум-подписки сервер вернёт «премиум-плейсхолдеры», и плагин уведомит об этом.
- **Капча Cloudflare**: если HDREZKA включит капчу для вашего IP, авторизация через API не сработает — войдите в браузере один раз, потом повторите попытку.
- **iOS Safari**: третьесторонние cookies могут блокироваться. Используйте CORS-прокси, который сам прокидывает cookie.
